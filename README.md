# Daggerheart Fear Tracker Plus

**Daggerheart Fear Tracker Plus** is a Foundry VTT module that provides extensive customization options for the Foundryborne Daggerheart system's Fear Tracker. Personalize your game's atmosphere with custom icons, color themes, and sizing options.

## Installation

1.  Open Foundry VTT and go to the **Add-on Modules** tab.
2.  Click **Install Module**.
3.  Paste the following Manifest URL into the field:
    ```
    https://github.com/cptn-cosmo/dh-feartrackerplus/releases/latest/download/module.json
    ```
    or search for Fear Tracker Plus in the Foundry VTT Add-on Modules tab.
4.  Click **Install**.

## Usage

1.  Enable the module in your world's **Manage Modules** settings.
2.  Go to **Configure Settings** > **Daggerheart Fear Tracker Plus**.
3.  Adjust the settings to your preference. Changes apply immediately or upon closing the settings window.
